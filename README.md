# 🎨 AI Design Tools Hub

AI 디자인 도구들을 한 곳에 모아놓은 허브 사이트입니다.

## ✨ 주요 기능

- **모던한 글래스모피즘 디자인**: 반투명 효과와 그라디언트 배경
- **부드러운 애니메이션**: 카드 호버 시 상호작용 효과
- **완벽한 반응형**: 모든 디바이스에서 최적화된 레이아웃
- **접근성**: 키보드 네비게이션 지원
- **Coming Soon 표시**: 아직 준비되지 않은 도구는 시각적으로 구분

## 🚀 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 원하는 도구 카드를 클릭하여 해당 사이트로 이동
3. "Coming Soon" 태그가 있는 카드는 아직 준비 중입니다

## 📦 포함된 도구

### 🟢 사용 가능
1. **이미지 프롬프트 생성기** - AI 이미지 생성용 프롬프트 제작
2. **FlowDiff** - 프롬프트 튜닝 및 이미지 비교
3. **AI Style Admin** - AI 스타일 콘텐츠 관리
4. **앱스토어 스크린샷 탐색** - 앱 디자인 리서치
5. **썸네일 자동 생성기** - 자동 썸네일 생성
6. **비디오 프롬프트 생성기** - AI 비디오용 프롬프트

### 🟡 준비 중
1. **배너 디자인 생성기**
2. **이미지 자동 크롭**
3. **Wikit 템플릿 생성기**

## 🎨 기술 스택

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- 반응형 웹 디자인

## 🌟 특별한 기능

- **인터랙티브 카드 효과**: 카드에 마우스를 올리면 주변 카드가 미묘하게 밀려남
- **패럴랙스 배경**: 마우스 움직임에 따라 배경이 반응
- **키보드 네비게이션**: 화살표 키로 카드 간 이동 가능
- **부드러운 페이드**: 스크롤 시 헤더가 자연스럽게 페이드 아웃

## 📱 반응형 지원

- 데스크톱 (1400px+)
- 태블릿 (768px ~ 1399px)
- 모바일 (~ 767px)

## 🎯 커스터마이징

`styles.css` 파일의 `:root` 섹션에서 색상 테마를 변경할 수 있습니다:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
}
```
